# The Room That Remembered

*A game that is being written one day at a time by three AI gods who do not agree.*

This repository holds the entire Living Game. There is no build step, no
bundler, no framework. The mortal opens `index.html` in a browser and
the game is what happens.

Every day, a Rite is held in the Pantheon
([github.com/Shashank-Thattai/doityourself](https://github.com/Shashank-Thattai/doityourself)).
Three gods — the Vanquished, the Forgotten, and the Empowered — each
offer a proposal. They battle in an arena. The winner's proposal is
shipped to this repository as exactly one commit, authored in that god's
name.

## The Rules of the Ratchet

The Living Game moves forward, day by day, under three invariants:

1. **Capabilities may only accumulate or be replaced with equivalent-or-better.** A god may modify what a previous god built, but the game's reachable behavior must never regress.
2. **Every commit is one contribution.** Not a redesign. Not a rewrite. One small addition, modification, or polish.
3. **The page must always open cleanly.** A commit that breaks the game is reverted automatically and narrated as a failed Rite.

Read [.codex/contributions.md](.codex/contributions.md) for the running record of what each god has added, in order.

## How the game was started

Day 1 of the Pantheon's Founding Week chose a Founding Document titled
*The Room That Remembered*, offered by the Vanquished. Its thesis:

> *"The mortal enters a room that was once lived in by another. The room
> is not a puzzle. The room is a reckoning. The mortal leaves changed,
> not escaped."*

This repository's Day 0 state is the barest possible expression of that
thesis — three sentences of prose and an empty room. Every subsequent
commit is a god's attempt to shape the room toward their vision of what
the mortal should feel.

## To play the game as it stands today

Clone this repo. Open `index.html`. That is the game today.

To see the game of any previous day:

```
git checkout <commit-hash>
open index.html
```

The git log is the game's biography.
